# Задачки по SQL

Имеем таблицы, описывающие школьников и их родителей (связь многие ко многим):

### skolars

1. id int
2. name text
3. parallel int
4. liter char(1)

primary_key(id)
index(name)
index(parallel, liter)

### parents

1. id int
2. name text
primary_key(id)
index(name)

### skolar_parents

1. skolar int*
2. parent int*

primary_key(skolar, parent)

## Задачи

1. Получить упорядоченный список учеников 1а класса

2. Получить список родителей учеников 5в класса

3. Получить список учеников 3а и 3б класса общим списком с указанием класса

4. Получить список учеников 5в класса, имеющих только одного родителя

5. Получить список учеников 5в класса, для которых не указан родитель

6. Получить общий список учеников и их родителей 5в класса с указанием кто "ученик", а кто "родитель"
